## Malware Types & Behavior Analysis (Basic)

### Objective
To understand different types of malware, their behavior, detection methods, and basic prevention techniques using malware analysis platforms.

---

### Concepts Covered

#### What is Malware
Malware is malicious software designed to damage systems, steal data, or disrupt operations without user consent.

---

#### Types of Malware
- **Virus:** Attaches to files and spreads when executed
- **Worm:** Self-replicates and spreads without user action
- **Trojan:** Disguised as legitimate software
- **Ransomware:** Encrypts data and demands payment

---

#### Malware Analysis Using VirusTotal
Malware file hashes were analyzed using VirusTotal to observe:
- Detection ratio
- Malware family classification
- Behavioral indicators

---

#### Malware Behavior Indicators
- File system modification
- Network communication with suspicious servers
- Persistence mechanisms
- Process injection

---

#### Malware Lifecycle
1. Delivery
2. Execution
3. Persistence
4. Command & Control
5. Payload execution

---

#### Malware Spread Methods
- Phishing emails
- Malicious downloads
- Infected USB devices
- Software vulnerabilities

---

#### Malware Prevention Techniques
- Use antivirus software
- Keep systems updated
- Avoid suspicious downloads
- Enable firewalls
- User awareness

---

### Outcome
This task improved understanding of malware classification, behavior analysis, and basic detection methods.
